## 🛠️ Skill yang Dipelajari

- Menyusun struktur website menggunakan Semantic HTML.
- Menggunakan Relative Path untuk mengakses aset.
- Membuat Form HTML dengan validasi dasar.
- Membuat Table HTML menggunakan `<thead>` dan `<tbody>`.
- Menampilkan gambar menggunakan `<img>`.
- Menampilkan audio dan video menggunakan `<audio>` dan `<video>`.
- Mengintegrasikan Google Maps menggunakan `<iframe>`.
- Mengintegrasikan video YouTube menggunakan `<iframe>`.
- Menerapkan atribut `alt`, `title`, dan `required` untuk meningkatkan kualitas HTML.
