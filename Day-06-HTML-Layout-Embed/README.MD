## 📚 Materi yang Dipelajari

Pada hari keenam, saya mempelajari cara membuat layout halaman web yang lebih lengkap serta menampilkan konten dari layanan eksternal menggunakan HTML.

Materi yang dipelajari meliputi:

- Tag `<iframe>`
- Embed Website

- Embed YouTube
- Relative Path
- Struktur Layout Website
- Accessibility pada iframe (`title`)

## 🛠️ Skill yang Dipelajari

- Menggunakan tag `<iframe>` untuk menampilkan konten eksternal.
- Mengintegrasikan Google Maps ke dalam website.
- Mengintegrasikan video YouTube ke dalam website.
- Menggunakan Relative Path untuk mengakses aset dari folder lain.
- Menyusun layout halaman menggunakan Semantic HTML.
- Menerapkan atribut `title` pada iframe untuk meningkatkan aksesibilitas.
