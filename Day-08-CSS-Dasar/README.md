# Day 8 - CSS Dasar

## 📚 Materi yang Dipelajari

Pada hari kedelapan, saya mulai mempelajari dasar-dasar CSS (Cascading Style Sheets) untuk mengubah tampilan halaman web yang sebelumnya dibuat menggunakan HTML.

Materi yang dipelajari meliputi:

- Apa itu CSS
- External CSS
- CSS Selector
- Color
- Background Color
- Font Family
- Font Size
- Text Align
- Padding
- Margin
- Border
- Border Radius
- Hover Effect
- Transition
- Transform

---

## 🎯 Mini Project

Membuat halaman sederhana dengan CSS yang menerapkan:

- Background halaman
- Styling pada Heading
- Hover Effect
- Animasi Transition
- Transform Scale

---

## 🛠️ Skill yang Dipelajari

- Menghubungkan file HTML dengan CSS menggunakan External CSS.
- Menggunakan Selector untuk mengatur tampilan elemen.
- Mengubah warna teks dan warna latar belakang.
- Mengatur jenis dan ukuran font.
- Memahami CSS Box Model (Margin, Padding, Border).
- Membuat sudut elemen menjadi membulat menggunakan Border Radius.
- Membuat efek interaktif menggunakan Hover.
- Membuat animasi halus menggunakan Transition.
- Memberikan efek pembesaran menggunakan Transform.

## 💡 Catatan

Pada materi ini saya mulai memahami bahwa HTML digunakan untuk membangun struktur website, sedangkan CSS digunakan untuk mengatur tampilan website.

Saya juga mulai memahami konsep CSS Box Model yang terdiri dari:

- Content
- Padding
- Border
- Margin

Selain itu saya mempelajari efek interaktif menggunakan Hover, Transition, dan Transform.
