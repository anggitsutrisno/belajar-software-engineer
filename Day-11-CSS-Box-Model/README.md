# Day 11 - CSS Box Model

## 📚 Deskripsi

Pada Day 11 saya mempelajari konsep **CSS Box Model**, yaitu dasar bagaimana setiap elemen HTML memiliki ukuran, jarak, dan batas.

Materi ini sangat penting karena hampir seluruh layout website dibangun menggunakan konsep Box Model.

- Width
- Height
- Padding
- Margin

- Border Radius
- Box Sizing
- Overflow

---

# 📖 Penjelasan Materi

## Width

Mengatur lebar elemen.

```css
width:300px;
```

---

## Height

Mengatur tinggi elemen.

```css
height:150px;
```

---

## Padding

Memberikan jarak antara isi dengan border.

```css
padding:20px;
```

---

## Margin

Memberikan jarak antar elemen.

```css
margin:20px auto;
```

---

## Border

Memberikan garis pada elemen.

```css
border:2px solid royalblue;
```

---

## Border Radius

Membuat sudut elemen menjadi melengkung.

```css
border-radius:10px;
```

---

## Box Sizing

Mengontrol perhitungan ukuran elemen.

```css
*{
    box-sizing:border-box;
}
```

---

## Overflow

Mengatur tampilan isi ketika melebihi ukuran elemen.

```css
overflow:auto;
overflow:hidden;
overflow:scroll;
overflow:visible;
```
