# Day 5 - HTML Multimedia

## 📚 Materi yang Dipelajari

Pada hari kelima, saya mempelajari cara menambahkan elemen multimedia ke dalam halaman website menggunakan HTML.

Materi yang dipelajari meliputi:

- Tag `<img>`
- Atribut `src`
- Atribut `alt`
- Atribut `width` dan `height`
- Relative Path
- Struktur folder multimedia
- Tag `<figure>`
- Tag `<figcaption>`
- Tag `<audio>`
- Tag `<video>`

---

## 🎯 Mini Project

Membuat halaman sederhana **Nerazurra Dev Studio** yang menampilkan berbagai elemen multimedia, yaitu:

- Logo perusahaan
- Keterangan gambar
- Audio profil perusahaan
- Video profil perusahaan

---

## 🛠️ Skill yang Dipelajari

- Menampilkan gambar menggunakan HTML.
- Menggunakan Relative Path untuk memanggil file.
- Mengelola aset multimedia dalam folder (`images`, `audio`, `videos`).
- Memberikan deskripsi gambar menggunakan `<figcaption>`.
- Menampilkan audio menggunakan `<audio>`.
- Menampilkan video menggunakan `<video>`.
- Memahami struktur folder proyek yang lebih rapi.

---
